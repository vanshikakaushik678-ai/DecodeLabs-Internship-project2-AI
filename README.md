# AI Classification Dashboard

An AI-powered web application that classifies data using Machine Learning models and provides an interactive dashboard for prediction and analysis.

## 📌 Project Overview

This project is developed as part of the Decode Labs Internship Program. The application allows users to upload data, perform classification using trained machine learning models, and visualize prediction results through an intuitive dashboard.

## 🚀 Features

* User-friendly web interface
* Machine Learning based classification
* Data upload support
* Real-time predictions
* Interactive dashboard
* Model performance visualization
* Responsive design

## 🛠️ Technologies Used

### Backend

* Python
* Flask

### Machine Learning

* Scikit-learn
* Pandas
* NumPy

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/vanshikakaushik678-ai/DecodeLabs-Internship-project2-AI.git
cd DecodeLabs-Internship-project2-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

## 📊 Workflow

1. Upload dataset.
2. Preprocess data.
3. Train or load classification model.
4. Generate predictions.
5. Visualize results on dashboard.
